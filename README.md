## English

A real-time cryptocurrency price display overlay designed for OBS Studio browser source. Features macOS-style window design, live sparkline charts based on 24-hour price history, and smooth color-coded price animations.

### Features

- **Real-time Data**: Fetches live prices from Binance Futures API every 5 seconds
- **24h Sparkline**: Displays the past 24 hours of price movement using 15-minute K-line data, with the latest point updated in real-time
- **macOS Style**: Clean window design with traffic light buttons
- **Live Indicator**: Green blinking "Live" badge in the corner
- **Price Animations**: Color changes (green/red) based on price direction with smooth transitions
- **Configurable**: Add or remove symbols via the config page
- **OBS Ready**: Works directly as an OBS browser source

### File Structure

### How to Use

1. Open `config.html` to add your preferred cryptocurrency symbols (e.g., `BTCUSDT,ETHUSDT,SOLUSDT`)
2. Open `index.html` in OBS as a browser source
3. Recommended OBS settings:
   - URL: Path to `index.html`
   - Resolution: 450x350
   - Custom CSS: `body { background-color: transparent; }` (optional)

### API Data Source

- **Real-time prices**: [Binance Futures API](https://developers.binance.com/docs/perpetual-futures/market-data/ticker) (`/fapi/v1/ticker/24hr`)
- **Historical data**: [Binance Futures K-line API](https://developers.binance.com/docs/perpetual-futures/market-data/kline-candlestick-data) (`/fapi/v1/klines`)

### Tech Stack

- Pure HTML/CSS/JavaScript (no build tools required)
- Binance Futures API (free, no API key required for public endpoints)
